Introduction to Programming

Programming languages are a set of commands and instructions that are written according to specific rules according to each language, and then these commands go through several stages until they are executed from the computer. But the computer, like any electrical device, only understands the language of the machine that adopts the binary number system (0, 1) corresponding to the two states (stopping, operating) ON - Off, which expresses the stopping or running of the machine. The computer does not understand any other human languages, and therefore the need to innovate and develop languages that the computer can understand has emerged. .

About HTML

Element

An element is one of the components of an HTML page. It represents semantics, or meaning. For example, the title element represents the title of a web page. Most HTML elements are written using the opening/start <> and closing/ending tags, with content in between. Elements can also have attributes that define their additional properties. For example, a paragraph will be written, which is represented by the element (p).

Attributes

Attributes specify additional properties such as the width and height of the image. Attributes are always placed in the starting tag (or opening tag) < > and are usually in the form of name-value pairs and are written in the following form: name = "value". Attribute values ​​should always be enclosed in quotation marks. Also, some attributes are necessary for certain components to function. For example, the < img > tag must contain the src and alt attributes.

Headings

Headings help define the hierarchy and structure of a web page's content. HTML provides six levels of heading tags, from < h1 > to < h6 > ; The higher the number in the heading, the more important it is - so the < h1 > tag defines the most important heading, while the < h6 > tag defines the least important heading on a web page. By default, browsers display headings in a font that is larger and darker than normal text. Also, < h1 > headings are displayed in the largest font, while < h6 > headings are displayed in the smallest font.

paragraphs

Paragraphs are created using the < p> tag, which spreads the text to web pages. The paragraph tag is a very basic tag and is usually the first tag that we will need to publish text on web pages The < br> tag is used to insert a line break in a web page. The < hr> tag is used to create horizontal lines to separate sections of content on a web page. Like < br>, the < hr> element is also an empty element.

links

A link or hyperlink is a connection from one web resource to another. Links allow users to move seamlessly from one page to another, on any server, anywhere in the world. The link has two ends, called anchors. The link starts at the Source point and points to the Destination point, which could be any web resource, for example, an image, audio clip, video clip, PDF file, HTML page, an element within the page itself, and so on.
